# personal-expenses-API

## Description
A RESTful API for managing personal expenses, including income and expense tracking.

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL (Neon)
- JWT Authentication

## Setup & Installation
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
4. Run migrations
5. Start the development server

## Environment Variables
List required environment variables in a `.env` file:
- `SECRET_KEY`
- `DATABASE_URL`
- `DEBUG` (optional)
